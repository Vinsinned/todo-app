# todo-app
###### Design inspired by [Todoist](https://www.todoist.com) and [Microsoft Todo](https://todo.microsoft.com/)
This project is a simple todo app has every basic features a todo app should have, including:
* Ability to CRUD todos
* Categories to sort todos
* Genres to group todos
* Instances to identify todos
* Ability to search todos
* Ability to sign in and sign off
* Database to store information
* A simple design that is easy to navigate and view contents
* Deadlines 


Along with all of the normal features, I am also going to implement some other features, such as:
* Priorities
* Quick entries
* Pictures
* Reordering tasks (Drag and Drops)
* Animations
* Accessibility
* Different Views
* Testing
* WAI-ARIA
* Responsiveness
## Technologies used
This project is a standard todo app that is made with **Express.js** and is built on the **express application generator** skeleton which uses the MVC architectural pattern. I will also be using **React.js** with this project in order to create and use components to create UIs. For my CSS preprocessor, I decided to go with **SASS** as it is the most popular preprocessor by far and this project could help me get familiar with using preprocessors. I have decided to create a todo app as I believe that is is a very basic application and can help me practice and get better at all of the technologies I used, along with new programming concepts such as asyncs and components.
